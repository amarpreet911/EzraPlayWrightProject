# Ezra Playwright Project

Playwright-based UI and API test suite for appointment booking workflows, with a focus on automation, reliability, and privacy validation.

---

## 🤖 AI-Assisted Development

This project leverages AI-assisted development to accelerate delivery while maintaining strong quality standards.

---
### Integration
- **GitHub Copilot** 
- **Playwright MCP Server** — Automated element detection via `mcp.json`
- **MCP Instructions** — Standardized prompts via `mcp_instructions.md`

### Key Configuration Files

| File | Purpose |
|------|--------|
| `mcp.json` | MCP server configuration |
| `mcp_instructions.md` | Standardized AI query format |
| `.github/copilot/instructions.md` | Copilot guidelines |
